** How to Run Locally**
1. Clone this repo
2. Run `npm install`
3. Run `node task-0.js`
4. Visit `http://localhost:3000/me` to see the output

** Stack**
Node.js + Express + Axios

** Features**
- Returns a JSON response with:
  - Status
  - User info (email, name, stack)
  - Current UTC timestamp
  - Random cat fact from Cat Facts API
